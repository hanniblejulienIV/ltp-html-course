For this tutorial, you will need:

  • A Windows computer
  • Administrator access
  • Windows Vista or later

BEFORE YOU START

There are two steps you will need to complete to get up and running with Python on your Windows machine.

  1. Install the Python program on your computer.
  2. Add your Python installation to the system path.

The installation of Python is completed using the Python installer. However, depending on your configuration, you may need to add your Python installation to the system path manually after running the Python installer.

RUNNING THE PYTHON INSTALLER

1. To install the Python program on your computer, you can use the Python windows installer, available at https://www.python.org/. At the top of the screen, you should see several menu options. Go ahead and click Downloads:

pic1 goes here

The Python site should detect your system configuration automatically and display two options, one for Python 2, and another for Python 3. Note that these are the latest versions of Python, so the version numbers in the picture may be different than the version numbers used here.

2. Click the download button for Python 3. This will initiate the download.

pic2

3. Once the download is finished, a dialogue should come up asking if you want to run the installer. This may look different, depending on which browser and/or version of Windows you are using. When this dialogue comes up, select run. If the following dialogue doesn’t show up automatically, find the downloaded installer in your filesystem (this is usually in the ‘downloads’ folder) and double click it.

pic3

4. When you run the installer, a new dialogue will come up. Depending on your configuration, there may be a checkbox allowing you to add Python to your system path. If this checkbox is available, go ahead and check it. If this check box is not available, don’t worry. I will later cover how to add the Python installation to the system path manually.

pic4

5. Select custom installation.

pic5

6. Click next until you arrive at a dialogue with a text field that says something like “customize install location”.

pic6

7. For most applications, the installation location doesn’t matter. However, for Python you may need to reference the install location. Therefore, making the installation directory simple, easy to find and easy to remember can be very helpful. In the “customize install location” field, enter in ‘C:/python3’

pic7

This will specify the base directory of the windows filesystem as the install location for Python.

8. You should click install. You may need to click next one or more times, and you may need to enter your password or verify that you would like to make changes.

pic8

9. When the installation process is finished, you should see a window indicating that the installation was successful. At this point, Python should be installed on your computer. You can now close this window.

pic9

CHECKING IF THE PYTHON INSTALLATION WAS ADDED TO THE SYSTEM PATH

1. Open the command prompt program on your computer. Command prompt is the windows command line interface which comes with every installation of Windows. If command prompt was previously open, close command prompt and open it again before continuing.

pic1.1

2. Type the word python into the command prompt and press enter. If your program was installed successfully and was added to the system path, you should see a new prompt with “>>” characters like this:

pic2.1

If your Python installation was not added to the system path, you should see a message which says that the python command is not recognized:

pic3.1

If you see a message indicating that the Python command is not recognized, follow the instructions to add the Python installation to the system path manually.

ADDING YOUR PYTHON INSTALLATION TO THE SYSTEM PATH MANUALLY

1. Open up the control panel on your computer. Click view by and select small or large icons. Select the system icon.

pic1.2

2. Select advanced system settings from the side panel.

pic2.2

3. Select the advanced tab at the top of the window. Click the environment variables button at the bottom of the screen.

pic3.2

4. In the environment variables section, select PATH and click edit.

pic4.2

5. Click new. In the new field that shows up, type the custom location that you set for your Python installation, which should be ‘C:/python3’.

pic5.2

6. Click OK. You have now added Python to the system path. You should then follow the previous instructions to verify that the Python installation is added to the path variable.
